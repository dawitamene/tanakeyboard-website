FEATURES

Everything Addiyon Keyboard does

A native Android keyboard designed around the Ge'ez script: accurate, fast, and completely on-device.

ሀ

Complete Amharic support

Every Fidel letter in all seven orders, plus every Amharic special character: punctuation (፣ ። ፤ ፥ ፦), the word separator ፡, and all labialized forms (ቈ ኰ ጐ ኈ). The full Ge'ez syllabary, nothing missing.

፲

All Ge'ez numbers

The complete set of Ethiopic numerals, ፩ ፪ ፫ … ፲ ፳ … ፻ ፼, for writing dates, prices and lists the traditional way.

✓

Robust, context-aware autocomplete

Frequency-ranked suggestions from large built-in dictionaries, smart about ambiguous cases: england → England, us (pronoun) vs US, march vs March, keeping proper nouns proper while respecting what you typed.

⌨️

Live Latin → Fidel transliteration

Type the sound and get the script. The whole word is re-read on every keystroke so it stays correct as you go, shown as an underlined composing preview until you commit.

📚

Huge offline dictionaries

~182,000 Amharic words and ~250,000 English words, all on-device, so suggestions and lookups work with no connection.

🔤

English layout included

A standard QWERTY English keyboard is built in, so you can switch between Amharic and English with a single tap.

🎨

Beautiful themes

Multiple keyboard color themes with automatic light and dark support, so it always fits your phone.

📱

Works everywhere · Private by design

A system keyboard that works in every app, and everything you type is processed on-device. See the privacy policy.
